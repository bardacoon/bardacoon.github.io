---
title: "How to Open/Close <Dialogs> without JS using [command] and [commandfor]"
tags: ['css']
pubDate: "Jun 11 2025"
---

Chrome introduced a new feature, the `command` attribute, that can be [applied to buttons](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/button#command).

The `command` takes in some predefined values, and you target the element you want to run this commands to with the `commandfor` attribute

```html
<button
    command="show-modal"
    commandfor="mydialog"
>
    Open Dialog
</button>

<dialog id="mydialog"></dialog>
```
